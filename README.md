# Тестовое задание: сервис скачивания и анализа файлов
Тестовое задание: сервис скачивания и анализа файлов  Перед вами API, которое раздаёт каталог текстовых файлов. Ваша задача — написать собственный сервис с простым веб-интерфейсом, который скачает весь каталог через это API и посчитает статистику по содержимому файлов.
